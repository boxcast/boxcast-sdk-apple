# BoxCast for Apple Platforms

The official BoxCast SDK for integrating with the BoxCast API on Apple platforms.

## Features

- [x] List Live and Archived Broadcasts
- [x] Detail A Broadcast
- [x] Play Live and Archived Broadcasts
- [x] Documentation

## Requirements

- iOS 9.0+
- Xcode 8.2+
- Swift 3.0+
